# Intents

Create intent folders using the format `{NNN}-{intent-name}`.

Each intent should include:

- `requirements.md`
- `system-context.md`
- `units.md`
- `inception-log.md`
- `units/{UUU}-{unit-name}/unit-brief.md`
- `units/{UUU}-{unit-name}/stories/{SSS}-{title}.md`
