# Unit Brief: favorites-and-outbound

Status: planned
Created: 2026-05-19T10:45:00Z

## Purpose

Let visitors express intent by saving stays and continuing to an external booking destination.

## Scope

Included:

- Favorite toggle on list and detail views.
- Favorites overview or section.
- Local favorite state.
- External booking CTA with clear labeling.

Excluded:

- User accounts.
- Server-side persistence.
- Payments.
- Email flows.

## Quality Bar

- Favorite toggle state is accessible by keyboard and screen reader users.
- Boolean state changes use explicit comparisons.
- External booking labels make it clear that the visitor leaves or continues outside the MVP.
