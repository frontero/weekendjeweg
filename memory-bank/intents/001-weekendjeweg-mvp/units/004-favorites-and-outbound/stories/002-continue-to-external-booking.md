# Story 002: Continue to External Booking

Status: planned
Created: 2026-05-19T10:45:00Z

## User Story

As a visitor, I want a clear way to continue to booking, so that I know when the app sends me to an external destination.

## Acceptance Criteria

- Given a stay has an external booking URL, when the visitor views the stay, then a clear outbound CTA is visible.
- Given the visitor activates the CTA, when navigation happens, then the label or surrounding copy makes clear that booking continues externally.
- Given a stay has no external booking URL, when the CTA area renders, then the app shows a safe fallback instead of a broken link.

## Notes

The MVP does not process payments or reservations.
