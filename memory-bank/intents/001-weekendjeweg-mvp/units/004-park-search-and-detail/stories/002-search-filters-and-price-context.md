# Story: Search Filters and Price Context

Status: planned
Unit: 004-park-search-and-detail

## User Value

As a visitor, I want to filter by region, dates, group size, and facilities so I can compare parks that fit my trip.

## Acceptance Criteria

- Filters include region, arrival date, departure date, adults, children, and dynamic facilities.
- Filter controls have accessible names and keyboard support.
- Result count updates are understandable to screenreader users.
- Price display includes context for selected dates and persons.
- UI text shows price only and never implies guaranteed availability.

## Notes

Facilities are dynamic from Landal data or mock data shaped like Landal-derived data.
