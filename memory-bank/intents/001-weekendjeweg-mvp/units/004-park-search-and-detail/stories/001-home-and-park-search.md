# Story: Home and Park Search

Status: planned
Unit: 004-park-search-and-detail

## User Value

As a Dutch visitor, I want to quickly find Landal parks for a weekend away.

## Acceptance Criteria

- `/` presents the usable Weekendjeweg discovery experience and links clearly to search.
- `/parken` lists parks from the data access layer or mock data.
- Search result cards are scannable and accessible.
- Empty states provide a reset path.
- Copy is Dutch and warm without being misleading.

First-release visuals use CSS/placeholder visuals; real images are deferred.
