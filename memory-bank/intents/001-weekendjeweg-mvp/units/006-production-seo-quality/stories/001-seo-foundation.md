# Story: SEO Foundation

Status: planned
Unit: 006-production-seo-quality

## User Value

As the project owner, I want the first release to be discoverable in search without making unsupported claims.

## Acceptance Criteria

- Home, park search, and park detail pages have accurate Dutch titles and descriptions.
- Sitemap support exists.
- Robots behavior is configured.
- Structured data is added only where accurate.
- No structured data or copy implies availability without approved data support.

## Notes

Rich region landing pages are deferred even though `/regio/[slug]` route support exists.
