# Story: Quality Gates

Status: planned
Unit: 006-production-seo-quality

## User Value

As the project owner, I want the MVP to be production-ready, tested, and measurable before release.

## Acceptance Criteria

- Typecheck and tests run successfully before release.
- Core logic tests cover normalization, filters, price selection, affiliate URL building, consent tracking, and compliance gating.
- E2E coverage includes home to search to detail to outbound CTA.
- Lighthouse checks target 90+ for Performance, Accessibility, and SEO on key pages.
- Remaining risks or blocked checks are documented before release.

## Notes

Lighthouse must be verified against the implemented app, not assumed from code structure.
