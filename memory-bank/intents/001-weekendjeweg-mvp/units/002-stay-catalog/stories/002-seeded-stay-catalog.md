# Story 002: Seeded Stay Catalog

Status: planned
Created: 2026-05-19T10:45:00Z

## User Story

As a visitor, I want a varied set of example stays, so that I can meaningfully browse and filter weekend options.

## Acceptance Criteria

- Given the MVP uses seed data, when the catalog loads, then it includes multiple regions and tags.
- Given a stay has an external booking URL, when the URL is missing or invalid, then the UI can present a safe fallback.
- Given the seed catalog is used in tests, when filters run, then test data covers matching and nonmatching results.

## Notes

Use realistic Dutch content, but avoid claiming real availability or prices.
