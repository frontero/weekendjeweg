# Story: GA4 Consent Banner

Status: planned
Unit: 005-affiliate-consent-tracking

## User Value

As a visitor, I want a clear cookie choice and no analytics loaded before I accept it.

## Acceptance Criteria

- Consent banner is keyboard accessible and screenreader understandable.
- Analytics accept and reject actions are explicit.
- GA4 is not loaded before analytics acceptance.
- Rejecting analytics does not break search, detail pages, or outbound CTAs.
- Consent state is respected on later page views.

## Notes

Consent UX should stay simple in the first release.
