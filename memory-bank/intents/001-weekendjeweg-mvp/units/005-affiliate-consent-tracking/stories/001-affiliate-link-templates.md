# Story: Affiliate Link Templates

Status: planned
Unit: 005-affiliate-consent-tracking

## User Value

As the project owner, I want Landal outbound links to be generated through templates so future affiliate parameters can be added cleanly.

## Acceptance Criteria

- Affiliate link templates are stored or represented through a reusable data model.
- URL builder accepts park context and future tracking parameters.
- Placeholder links are clearly separated from final network configuration.
- The CTA can use the generated URL without duplicating URL-building logic.

## Notes

Final Landal affiliate-network mapping is deferred until account/network details are known.
