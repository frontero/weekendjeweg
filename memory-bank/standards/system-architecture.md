# System Architecture Standard

Status: pending.

Architecture decisions must be documented before construction when they affect shared contracts, deployment, persistence, routing, authentication, authorization, or data ownership.

## Current Principles

- Keep source of truth in one place.
- Prefer established framework patterns over custom abstractions.
- Use typed contracts for shared data structures.
- Keep user-facing workflows accessible and testable.
- Record significant decisions in `memory-bank/standards/decision-index.md`.
